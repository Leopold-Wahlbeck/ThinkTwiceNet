# ThinkTwiceNet
A staged ensemble framework combining heterogeneous models with confidence-based boosting for adaptive inference.


## Structure

- `src/` code
- `configs/` experiment configs
- `docs/` planning and experiment logs
- `data/` datasets (not tracked in git)
- `outputs/` models, logs, predictions (not tracked in git)

## First goal

Build:
1. Stage 1 simple classifier
2. Confidence threshold
3. Stage 2 CNN
4. Evaluation pipeline
